Dies ist eine allgemein lesbare Zusammenfassung der ODbL 1.0 Lizenz. Bitte beachten Sie auch die untenstehenden Haftungshinweise.

Sie dürfen:

  • Weitergabe: Kopieren, Verteilen und Nutzen der Datenbank.
  • Erschaffen: Von der Datenbank abgeleitete Werke herstellen.
  • Anpassen: Ändern, Umformen und Erweitern der Datenbank.

Zu den folgenden Bedingungen:

  • Kennzeichnung: Jede öffentliche Nutzung der Datenbank oder von ihr abgeleiteter Werke muss entsprechend der Vorgaben in der ODbL gekennzeichnet werden. Bei jeglicher Nutzung oder Neuverteilung der Datenbank, oder abgeleiteter Werke, muss für andere die Lizenz der Datenbank erkennbar gemacht werden. Sämtliche Hinweise auf die Original-Datenbank müssen erhalten bleiben.
  • Weitergabe unter gleichen Bedingungen: Wenn eine angepasste Version dieser Datenbank, oder von einer solchen abgeleitete Werke, öffentlich verwendet werden müssen diese angepassten Datenbanken wiederum als ODbL lizensiert werden.
  • Erhaltung der Offenheit: Wird die Datenbank, oder eine angepasste Version dieser, weiterverteilt, so können hierfür technologische Mittel genutzt werden, welche die Arbeit einschränken (wie z.B. DRM), solange auch immer eine Version ohne diese Einschränkungen mitverteilt wird.
